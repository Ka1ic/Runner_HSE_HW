# Runner_HSE_HW

ФИО: Калицов Амиран Иналович
Краткое описание: В игре одна сцена, при запуске сразу же начинается забег. Большая часть функционала игры содержится в скрипте PlayerController.cs, но большая часть скриптов сообщаются через объект со скриптом GameManager.cs (думаю, его можно назвать главным). 
В игре есть возможность бежать по трём полосам уклоняясь от препядствий благодаря перемещениям с полосы на полосу и прыжкам. Игра со временем ускоряется, пока не достигает максимальной скорости. По ходу игры ведётся учёт очков в зависимости от пройденного расстояния, при проигрыше обнуляется. Присутствует 3 здоровья, при столкновении с серым объектом -1 здоровье, с красным -2 здоровья, при достижении 0 здоровья забег заканчивается, есть возможность начать новый. Присутствуют бонусы в виде восполнения сердец (не более 3), и временной неуязвимости (3 секунды). Присутствует несколько различных частей дороги с препядствиями, они создаются по ходу игры в рандомном порядке. Управления по средствам клавиш wad или стрелки + можно использовать пробел для прыжка. Управление реализовано на новой системе ввода Unity. При столкновении с игроком, игрок получает урон (если он не неуязвим в данный момент), а препятствие взрывается и игрок продолжает движение, если он не умер от столкновения. Так же, для красоты настроил партиклы разрушения припядствий и игрока + настроил Cinemachine для красивого слежения за игроком + настроил UI c TextMeshPRO.
